# Space Explorer Webpage

**An Interactive Space-Themed Portfolio Page**

---

## Overview

The **Space Explorer** webpage is a visually stunning, highly interactive, and creatively designed website that simulates a journey through space. Featuring animated stars, orbiting planets, moving comets, and a launchable spaceship, this project demonstrates advanced HTML, CSS, and JavaScript techniques. It's perfect for impressing potential employers and fellow developers during interviews or portfolio presentations.

---

## Features

- **Animated Starfield Background:** Dynamic, twinkling stars that create a deep space atmosphere.
- **Orbiting Planet:** A planet that smoothly orbits around the center of the viewport.
- **Moving Comet:** A comet that streaks across the sky with realistic motion.
- **Interactive Celestial Bodies:** Clickable planets and comets with alert messages.
- **Spaceship Launch Animation:** Click the button to launch the spaceship upwards with smooth animation.
- **Stylish Design:** Neon glow effects, smooth hover animations, and responsive layout for desktop.

---

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling and animations
- **JavaScript ES6+** for interactivity and animations
- **Canvas API** for starfield background

---

## Setup & Usage

### Clone or Download

1. Download the ZIP archive or clone the repository:

```bash
git clone https://github.com/JCaesar45/space-explorer.git
``

2. Navigate into the project directory:

```bash
cd space-explorer
``

### Open in Browser

Simply open the `index.html` file in your preferred web browser to view the webpage:

```bash
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
``
